### Getting Started

#### Clone the project

```
git clone https://github.com/AllefDouglasDev/sale-manager-back.git
```

#### Go to the project folder

```
cd sale-manager-back
```

#### Install all dependencies

```
yarn
```

#### Run the project

```
yarn dev
```

#### Build for production

```
yarn build
```

#### Run in production

```
yarn start
```
